================================================================================
TROUBLESHOOTING
================================================================================

Problem: "ModuleNotFoundError: No module named 'main'"
Solution: Make sure you run from xml_to_scl directory:
  cd C:\Projects\MODULBLOCK_MBK2\MBK_2\PLC_410D1\xml_to_scl
  python batch_convert_project.py "C:\..."

Problem: "No XML files found"
Solution: Verify source path contains .xml files:
  Get-ChildItem "C:\Path" -Recurse -Filter "*.xml" | Measure-Object

Problem: Report won't open in Excel
Solution: Use UTF-8 encoding when opening:
  1. Open Excel
  2. File → Open
  3. Select batch_conversion_report.csv
  4. Choose UTF-8 encoding

Problem: Console shows encoding errors
Solution: Set PowerShell encoding:
  chcp 65001

For more: See BATCH_CONVERTER_README.md troubleshooting section
